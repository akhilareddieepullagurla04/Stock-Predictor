# Stock Predictor 📈

## Description
This project predicts stock prices using Machine Learning.

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- yFinance

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Features
- Stock price prediction
- Data visualization
- Real-time stock data
